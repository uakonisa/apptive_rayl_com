Backend Theme Manon

Odoo Manon theme is backend appearance theme for Odoo.

The theme aims at giving a complete makeover to your Odoo backend.

The users can add virtually unlimited Font variations to the Odoo from backend.

Any number of color codes can also be added to change the color of buttons, forms, etc.

The backend theme is designed to be mobile responsive which means it auto-adjusts to all screen types such as tablets, mobile phones, computer screens, etc.

NOTE:

If you are looking for additional improvements apart from the default options, the Backend theme is completely customizable.
Raise a ticket with us for any query, suggestions and customization solutions.
