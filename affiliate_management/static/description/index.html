Odoo Affiliate Extension

You can easily manage your Affiliates

ODOO Affiliate Extension for ODOO E-Commerce store set commission for affiliates for each visitor or customer brought by the affiliate's own marketing efforts.

This Extension Allows

  • Real-time reporting.
  • Traffic Statistics.
  • Unlimited Affiliates.
  • Easy affiliate two step recruitment.
  • Affiliate approval and notifications.
  • Automatic invoicing for Traffic and Order record.
  • Disable Traffic record commission .
  • Enable the unique Traffic Report on product.
  • Set the different commission programs for Traffic and Order record .
  • Set the cookie expiration time for commission calculation .
  • Set your threshold amount in commission calculation .
  • Advance commission option .
  • Set different banner for advertisement.
  • Quick and Easy Installation.

  • Full Admin Control
    • Confirm the affiliate request manually and automaticaly
    • Define/Affiliate commission program as per their preferences .
    • Generate/Banner for Advertisement
    • Define/Update welcome page Banner and Title.
    • Define/Update Email Text sent to user for registration, welcome and cancel request.
  • Website View
  • Admin Side View
  • Configuration
